# Skills

Installed [skills.sh](https://skills.sh) skills live here, one folder per skill —
added during scaffolding, via `npx skills add <owner/repo[@skill]> --agent universal`,
or by agents themselves.

**Agents:** when a skill here matches the current task, read its instructions and
apply it. Each skill's own README/SKILL.md explains when and how to use it.

> Seeded empty by NEPTR — this README is the only file until a skill is installed.
